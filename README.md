# Retail_Sales_Analytics_Dashbaord

## Overview
An interactive Power BI dashboard developed to analyze retail sales performance across product categories including Clothing, Accessories, Bikes, and other retail products. The dashboard provides insights into sales trends, customer behavior, and business performance metrics.

## Key Features
- Sales performance KPIs
- Category-wise analysis (Clothing, Accessories, Bikes, etc.)
- Revenue and profit trends
- Interactive slicers and filters
- Regional and product-level insights
- Monthly and yearly sales analysis

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel
